What is a salt? A salt is a random string added to your password before hashing. It makes hashes stronger and prevents attackers from using precomputed lists.

How to Use This Hash in Your Website

You can store this hash instead of the real password. When a user logs in, hash their input and compare it.


// Example (JavaScript)
const storedHash = "PASTE_HASH_HERE";

async function verifyPassword(input) {
    const encoder = new TextEncoder();
    const data = encoder.encode(input);
    const hashBuffer = await crypto.subtle.digest("SHA-256", data);
    const hashArray = Array.from(new Uint8Array(hashBuffer));
    const hashHex = hashArray.map(b => b.toString(16).padStart(2, "0")).join("");

    return hashHex === storedHash;
}

This is how websites safely check passwords without ever storing the real one.
